Factorials

What is a Factorial?

! is the symbol used for factorial. Factorial of a number is the product of number and the numbers before it, For example factorial of 5 is 5*4*3*2*1 = 120.

Code For Factorial

import java.util.*;

public class pattern2

{

static void main()

{

int n,a;

long i,f=1,j;

double s=0,s1=0.0;

Scanner sc=new Scanner(System.in);

System.out.println( "Enter the end value");

n=sc.nextInt();

System.out.println( "Enter constant value");

a=sc.nextInt();

for(i=0 ;i<=n ;i++)

{

for(j=a ; j<=n ;j++)

{

f=f*a; s=s+f;

}

s1= s1+ (int)Math.pow(a,i)/s;

}

System.out.println(s1);

}

}

Here the outer loop i is for calculating the sum n number of times while the inner loop j finds the factorial of a.
