Niven Number

What is a Niven Number?

A number is said to be the Niven number if it is divisible by the sum of its digit.

Code For Niven Number

import java.util.*; public class Niven_Number { public static void main() {

Scanner in = new Scanner(System.in);

int num;

int rem = 0, sum = 0, n;

System.out.println("Enter the number to check");

num = in.nextInt();

n = num;

while(num > 0)

{

rem = num%10;

sum = sum + rem;

num = num/10;

}

if(n%sum == 0)

System.out.println(n + " is a Niven number");

else

System.out.println(n + " is not a Niven number"); } }

In this program we are first defining and taking the value of the variable num from the user. Then we are making a copy of the num by storing the value of num in variable n. We use a while loop in which we divide the number by 10 to get its remainder i.e. its digit which is then added to our total sum. After the loop exits, variable n undergoes a condition, it checks whether the number is divisible but the sum or not. If it does then the given number is a Niven number. Else, given number is not a Niven number.
