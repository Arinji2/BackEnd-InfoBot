Brown Pair Number

What is a Brown Pair Number?

If the sum of the factor of the smaller number + 1 is equal to the square of the larger number.

Code For Brown Pair Number

------

import java.util.*;

public class brownpair

{

static void main()

{

int n1,n2,max,min,i,f=1;

Scanner ob=new Scanner(System.in);

System.out.println("Enter 2 numbers");

n1=ob.nextInt();

n2=ob.nextInt();

max=Math.max(n1,n2);

min=Math.min(n1,n2);

for(i=1;i<=min;i++)

{ f=f*i;

}

max=(int)Math.pow(max,2);

if((f+1)==max)

System.out.println("brown Pair");

else

System.out.println("Not a brown Pair");

}

}
