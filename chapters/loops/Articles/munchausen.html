Munchausen

What is a Munchausen?

Munchausen Number:-A Munchhausen number is a number equal to the sum of its digits raised to each digit's power.

Code For Munchausen

import java.util.*;

class munchausen

{

static void main()

{

Scanner ob = new Scanner(System.in);

int n;

System.out.println("Enter the number");

n=ob.nextInt();

int t,r,s=0;

t=n;

while(t>0)

{

r=t%10;

s=s+(int)Math.pow(r,r);

t=t/10;

}

if(n==s)

System.out.println("It is a munchausen number");

else

System.out.println( n+"is not a munchausen number");

}

}

In this program, we first take the input and make the copy of the input number in other variable.Then we use digit extraction technique and raise the digit to digit's power and store it in a variable.At the end we print appropiate message according to the final value obtained.
