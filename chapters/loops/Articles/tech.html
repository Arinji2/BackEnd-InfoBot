Tech Number

What is a Tech Number?

A number is called a tech number if the given number has an even number of digits and the number can be divided exactly into two parts from the middle. After equally dividing the number, sum up the numbers and the square should be equal to the original number

Code For Tech Number

import java.util.*;

public class program

{

public static void main()

{

int n, num, a, b, digits=0, square;

Scanner in = new Scanner(System.in);

System.out.println("Enter the number: ");

n = in.nextInt();

num = n;

while (num > 0)

{

digits++;

num = num / 10;

}

if (digits == 4)

{

num = n;

a = num % 100;

b = num / 100;

square = (a+b) * (a+b);

if (n == square)

{

System.out.println(n +" is a Tech number.");

}

else

{

System.out.println(n +" is not a Tech number.");

}

}

else

{

System.out.println(n + " is not a Tech number.");

}

}

}

In this program, We first check the number of digits in the input number using a while loop, if the number isn't even, the input number is not a Tech number else continue by splitting the number into 2 halves by dividing the number. after which we find the sum of the squares and check it with the original number
