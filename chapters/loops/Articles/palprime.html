Palprime

What is a Palprime?

The number should be a prime number as well as a palindrome number.A Palindrome Number is a number which remains the same when reversed. For Example: 121 , 454

Code For Palprime

import java.util.*;

class palprime

{

static void main()

{

Scanner ob=new Scanner(System.in);

int n;

System.out.println("Enter number");

n=ob.nextInt();

int t=n,k=0,s=0,r;

int i;

for(i=1;i<n;i++)

{

if(n%i==0)

k++;

}

while(t>0)

{

r=t%10;

s=(s*10)+r;

t=t/10;

}

if(s==n && k==1)

System.out.println("The number is a palprime number");

else

System.out.println("The number is not a palprime number");

}

}

For the Palprime number, we made a copy of the input number and first counted the number of factors of the number. .Then we reversed it using Character Extracting. Finally we checked whether it was a palprime number or not.
