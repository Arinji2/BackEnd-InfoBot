Greatest Number Amongst 10

What is a Greatest Number Amongst 10?

To input ten numbers and find the greatest number amongst them

Code For Greatest Number Amongst 10

import java.util.*;

public class program

{

public static void main()

{

Scanner in = new Scanner(System.in);

int num;

int i, max=0;

System.out.println("Enter ten numbers");

for(i=1;i<=10; i++)

{

num = in.nextInt();

if(num>max)

max = num;

}

System.out.println("The greatest number amongst them is "+max);

}

}

The program loops for 10 times through each number, accepting the number and checking it against the current max, and if the current number is greater than the current max, it sets it to the max.
