Special Number

What is a Special Number?

A special two-digit number is such that when the sum of its digits is added to the product of its digits, the result is equal to the original two-digit number. Example: Consider the number 59. Sum of digits => 5 + 9 = 14 Product of its digits => 5 x 9 = 45. Sum of the sum of digits and product of digits => 14 + 45 = 59

Code For Special Number

import java.util.*;

public class specialNo

{

static void main()

{

Scanner sg=new Scanner(System.in);

System.out.println("enter 2 digit number");

int n=sg.nextInt();

int k=0,r,s=0,p=1,t=n;

while(t>0)

{

r=t%10;

k++;

s=s+r;

p=p*r;

t=t/10;

}

if(k==2)

{

if((s+p)==n)

System.out.println(n + " is special 2 digit no");

}

else

System.out.println(n + " is not a special 2 digit no");

}

}

First check if the number entered is a two digit number. Sum and Product of digits is found using while loop by extracting last digit. Finally check if the sum of the, Sum and Product of digits is equal to the number itself.
