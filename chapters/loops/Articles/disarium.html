Disarium

What is a Disarium?

A number is called Disarium if sum of its digits powered with their respective positions is equal to the number itself.Example:- 89=8+(9*9)

Code For Disarium

import java.util.*;

class Disarium

{

static void main()

{

Scanner sc=new Scanner(System.in);

System.out.print("Input a number : ");

int num = sc.nextInt();

int copy = num, d = 0, sum = 0;

String s = Integer.toString(num);

int len = s.length();

while(copy>0)

{

d = copy % 10;

sum = sum + (int)Math.pow(d,len);

len--;

copy = copy / 10;

}

if(sum == num)

System.out.println("Disarium Number");

else

System.out.println("It is not a Disarium Number");

}

}

In this program, we take the input of a number and make a copy of it.Then using wrapper class we convert the number into a string. Then we use digit extraction and store the sum of the digits raised to its position.At the end we check whether the sum is equal to the input number.
