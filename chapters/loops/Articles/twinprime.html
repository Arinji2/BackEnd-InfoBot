TwinPrime

What is a TwinPrime?

Pair of prime numbers with a difference of two.

Code For TwinPrime

import java.util.*;

public class twinprime

{

static void main()

{

int n,n1, k=0,k1=0,i;

Scanner ob=new Scanner(System.in);

System.out.println("Enter the numbers");

n=ob.nextInt();

n1=ob.nextInt();

for(i=1;i<=n;i++)

{

if(n%i==0)

k++;

}

for(i=1;i<=n1;i++)

{

if(n1%i==0)

k1++;

}

if(k==2&&k1==2&&Math.abs(n1-n)==2)

System.out.println(n+" " + n1+" twin prime numbers ");

else

System.out.println(n+" " + n1+" is not twin prime numbers ");

}

}

Here first we check if the numbers are prime or not using for loop. Then we find the different between the two numbers. Math.abs is used to get a positive value for the difference.
