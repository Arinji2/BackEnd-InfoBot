Neon

What is a Neon?

Neon number is a number where the sum of digits of square of the number is equal to the number. (eg.9 9*9 = 81 and 8+1=9)

Code For Neon

import java.util.*; class neon { static void main() {

Scanner ob=new Scanner(System.in);

System.out.println("Enter a number");

int n=ob.nextInt();

int s=n*n;

int a=0,r;

while(s>0)

{

r=s%10;

a=a+r;

s=s/10;

}

if(a==n)

System.out.println("It is a Neon number");

else

System.out.println("It is not a Neon number"); } }
