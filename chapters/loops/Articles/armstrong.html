Armstrong Number

What is a Armstrong Number?

If the sum of the cubes of every digit in a number is equal to the number itself then it is called an Armstrong number For example: 153 = 13 + 53 + 33

Code For Armstrong Number

------

import java.util.*;

public class armstrong{

static void main(){

Scanner in = new Scanner(System.in);

int n,r,t,s=0;

System.out.println("Enter a number");

n=in.nextInt();

t=n;

while(t>0){

r=t%10;

s=s + (int)Math.pow(r,3);

t=t/10;

}

if(n==s)

System.out.println("It is a Armstrong number");

else

System.out.println("It is not a Armstrong number");

}

}

------

For Armstrong Number, we made a copy of the input number and raised the power of each digit it using Digit Extraction. Finally we checked if the number we calculated is equal to the input number.
