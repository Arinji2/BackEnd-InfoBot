Palindrome Number

What is a Palindrome Number?

A Palindrome Number is a number which remains the same when reversed. For Example: 121 , 454

Code For Palindrome Number

------

import java.util.*;

public class palindrome{

static void main(){

int num,n=0,r,s=0, store=0;

Scanner in = new Scanner(System.in);

System.out.println("Enter a number");

num=in.nextInt();

store=num;

while(n < num){

r=num%10;

s=s*10+r;

num=num/10;

}

if(s==store)

System.out.println("It is a palindrome number");

else

System.out.println("It is not a palindrome number");

}

}

------

For Palindrome Number, we made a copy of the input number and reversed it using Character Extracting. Finally we checked if the new reversed number is equal to the input number
