Code For Pattern

class pattern { static void main() {

int i,j;

for(i=1;i<=5;i++)

{

for(j=5;j>i;j--)

{

System.out.print(" ");

}

for(j=1;j<=i;j++)

{

System.out.print((i*j)+" ");

}

System.out.println();

} } }

We use 3 for loops. The first i loop we use for columns. Second loop we use to add spaces row wise. Third loop we use to print the numbers row wise.
