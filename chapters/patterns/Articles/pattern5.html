public class pattern

{

static void main()

{

int i,j;

int k=1;

for(i=1;i<=5;i++)

{

for(j=1;j<=i;j++)

{

k=k+2;

System.out.print(k+" ");

}

System.out.println();

}

}

}

Firstly we use the first for loop to print the number of rows. Second we use the next for loop to print the elements in the rows. The k variable is initialized to 1 and then added by 2 to print all the odd numbers
