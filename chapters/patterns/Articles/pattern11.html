Code For Pattern

class pattern5 { static void main() {

int i,j;

char b='A';

for(i=1;i<=5;i++)

{

for(j=1;j<=i;j++)

{

System.out.print(b++);

}

System.out.println();

} } }

i loop is used for columns and j loop is used for rows. We print the character with post increment.
