class Program

{

public static void main()

{

String str="BLUEJ";

int l=str.length();

for(int i=l-1;i>=0;i--)

{

for(int j=l-1;j<=i;j--)

{

System.out.print(str.charAt(i));

}

System.out.println();

}

}

}

The outer loop (i) that runs 5 times creates the shape. j, the outer loop is used to create the index at which the character will be extracted.
