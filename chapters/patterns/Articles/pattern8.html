Code For Pattern

class pattern { static void main() {

int i,j;

for(i=0;i<=10;i+=2)

{

for(j=1;j<=i;j++)

{

System.out.print(" ");

}

for(j=i;j<=10;j++)

{

System.out.print("*");

}

System.out.println();

} } }

We use 3 for loops in this program. i loop is used for columns. First j loop is used for spaces and second for stars.
