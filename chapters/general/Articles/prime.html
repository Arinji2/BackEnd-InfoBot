Prime Number

What is a Prime Number?

A Prime Number is a number with only 2 factors, that is the number itself and 1. For Example 2 -> 1,2

Code For Prime Number

import java.util.*;

public class prime

{

int factor(int n)

{

int i, k=0;

for(i=1; i<=n; i++)

{

if(n%i==0)

k++;

else

continue;

}

return(k);

}

void main()

{

Scanner ob=new Scanner(System.in);

System.out.println("Enter a number");

int x=ob.nextInt();

int p=factor(x);

if(p==2)

System.out.println(x+" is a prime number");

else

System.out.println(x+" is not a prime number");

}

}

Firstly we find the number of factors of the number input by the user in the factor(n) method and return it to the main method. In the main method, it is checked whether the number of factors is 2 or not
