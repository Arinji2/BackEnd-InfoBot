What is a Pattern?

Triangle to print the string pattern ICSE alphabet by alphabet.

Code For Pattern

public class pat4

{

static void main()

{

int i;

String str = "ICSE";

for(i=1;i<=4;i++)

{

System.out.print(str.substring(0,i));

System.out.println();

}

}

}

String function s.substring() is used that extracts a characters from a given index to i and prints it. For loop runs 4 times since ICSE contains 4 characters.
