Pattern.

What is a Pattern?

A triangle pattern to display 1 to 15 with row capacity increasing from 1 to 5.

Code For Pattern.

public class pat3

{

public static void main()

{

int k=1,i,j;

for(i=1;i<=5;i++)

{

for(j=1;j<=i;j++)

{

System.out.print(k+ " ");

k++;

}

System.out.println();

}

}

}

two for loops and a counter variable is used. Counter variable is printed and it increases from 1 to 15. i loop is used for columns. j loop is used for rows.
