What is a Series?

To find the sum with the use of for loop

Code For Series

import java.util.*;

public class series2

{

static void main()

{

int i;

double s=0.0;

for(i=5 ; i < = 10 ; i++)

s=s+ (i-1)/i;

System.out.println("Sum ="+s);

}

}

here the loop finds the sum for numbers 4/5 to 9/10
