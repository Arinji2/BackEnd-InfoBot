Series

What is a Series?

To find the sum with the use of for loop

Code For Series

import java.util.*;

public class series1

{

static void main()

{

int n,i; double a;

double s=0.0;

Scanner ob=new Scanner(System.in);

System.out.println("Enter the nth term");

n=ob.nextInt();

System.out.println("Enter constant value");

a=ob.nextDouble();

for(i=1 ; i <= n ; i+=2)

{

s=s+ a/i;

}

System.out.println("Sum ="+s);

}

}

a is used for the constant provided by the user while n is used for the number of terms for addition to find sum through for loop.
