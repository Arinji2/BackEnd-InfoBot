Series

What is a Series?

The counter variable (i) gets incremented by one after each iteration. If its value is even its square gets increased by 1 else it gets reduced by 1

Code For Series

import java.util.*;

public class Series {

static void main() {

int n,i;

Scanner in=new Scanner(System.in);

System.out.println("Enter the number of terms");

n=in.nextInt();

for(i=1;i<=n;i++){

if(i%2!=0){

System.out.print((int)(Math.pow(i,2))-1+" ");

}

else {

System.out.print((int)(Math.pow(i,2))+1+" ");

}

}

}

}

The counter variable (i) gets incremented by one after each iteration. If its value is even its square gets increased by 1 else it gets reduced by 1
