1/a! + a/a! + a^2/a! + a^3/a! +......n

What is a 1/a! + a/a! + a^2/a! + a^3/a! +......n?

raise a(the input number) to the power of 0,1,2,.. divided by factorial of a.

Code For 1/a! + a/a! + a^2/a! + a^3/a! +......n

import java.util.*;

public class pattern_1

{

static void main()

{

Scanner in=new Scanner(System.in);

System.out.println("Enter the number of terms");

int n=in.nextInt();

double sum = 0.0;

System.out.println("Enter the number");

int a = in.nextInt();

int i,f=1;

for(i=1;i<=a;i++)

{

f = f*i;

}

for(i = 1;i<=n; i++)

{

//System.out.print(a+ "^" + i + "/" + f + " + ");

sum = sum + (Math.pow(a,i))/f;

}

System.out.println("sum = " + sum);

}

}

Factorial of the entered number is found using for loop and stored in variable f. The next for loop is used to find the sum of the number raised to i (that updates in for loop) divided by f.
