Overloading Calculate

What is Overloading Calculate?

To overload the function calculate by using different parametric list.
(i) void calculate(int m, char ch) with one integer argument and one character argument. It checks whether the integer argument is divisible by 7 or not, if ch is 's', otherwise, it checks whether the last digit of the integer argument is 7 or not
(ii) void calculate(int a, int b, char ch) with two integer arguments and one character argument. It displays the greater of integer arguments if ch is 'g' otherwise displays the smaller of integer arguments.

Code For Program

import java.util.*;

public class overload

{

static void calculate(int m, char ch)

{

switch(ch)

{

case 's':

if(m%7==0)

System.out.println( m+" is divisible by 7");

else

System.out.println( m+" is not divisible by 7");

break;

default:

if(m/10==7)

System.out.println("the last digit of "+m+" is 7");

else

System.out.println( "the last digit of "+m+" is not 7");

}

}

static void calculate (int a, int b, char ch)

{

switch(ch)

{

case 'g':

int max= Math.max(a,b);

System.out.println("The greatest of the two integers is "+ max);

break;

default:

int min= Math.min(a,b);

System.out.println("The smallest of the two integers is "+ min);

}

}

static void main()

{

calculate(77,'s');

calculate(16,20,'b');

}

}

The Math. max() function returns the largest of the zero or more numbers given as input parameters while Math.min() function is an inbuilt function in java that returns the minimum of two numbers
