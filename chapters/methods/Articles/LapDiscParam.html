What is a laptop discount using parameterized constructor?

A class based program of finding the special discounts on laptops by using a parameterized constructer

Code For laptop discount using parameterized constructor

import java.util.*;

public class laptop

{

String name;

float price;

double dis,amt;

laptop (String n, float pr)

{

name = n;

price = pr;

}

public void compute()

{

if (price <= 25000)

dis = price * 0.05;

else if (price <= 50000)

dis = price * 0.075;

else if (price <= 100000)

dis = price * 0.1;

else

dis = price * 0.15;

amt = price - dis;

}

public void display()

{

System.out.println("Name: " + name);

System.out.println("Discount: " + dis);

System.out.println("Amount to be paid: " + amt);

}

public static void main()

{

Scanner in = new Scanner(System.in);

System.out.print("Enter Customer Name: ");

String str = in.nextLine();

System.out.print("Enter Price: ");

float p = in.nextFloat();

laptop ob = new laptop(str,p);

ob.compute();

ob.display();

}

}

In this program, the class name is named as laptop as mentioned in the question. Data members like name, price, dis and amt are initialized in the parameterized constructor. public void compute() method is used to find the discount and the total amount and is displayed in public void display(). In the main method, an object of the class is created and it calls the member methods.
