method:HCF & LCM

What is a method:HCF & LCM?

HCF of two or more numbers is the greatest factor that divides the numbers evenly.The LCM of Least Common Multiple of 2 numbers is the smallest number, divisible by the two numbers, evenly

Code For method:HCF & LCM

import java.util.Scanner;

public class Hcflcm

{

int a;

int b;

Hcflcm(int x, int y)

{

b = y;

a=x;

}

void calculate()

{

int hcf=0,lcm=0;

for(int i=1;i<=Math.min(a,b);i++)

{

if(a%i==0&&b%i==0)

hcf=i;

}

lcm=(a*b)/hcf;

System.out.println("HCF = " + hcf);

System.out.println("LCM = " + lcm);

}

static void main()

{

Scanner in = new Scanner(System.in);

System.out.print("Enter first number: ");

int x = in.nextInt();

System.out.print("Enter second number: ");

int y = in.nextInt();

Hcflcm obj = new Hcflcm(x,y);

obj.calculate();

}
