Successive Discounts

What is a Successive Discounts?

import java.util.*;

public class SuccessiveDiscount {

static int Discount(int a){

int d=(a*10)/100;

int dp=a-d;

return dp;

}

static int Discount(int a, int b) {

int d1=(a*b)/100;

int dp1=a-d1;

return dp1;

}

static void main() {

Scanner in=new Scanner(System.in);

int p;

System.out.println("Enter the price of the article");

p=in.nextInt();

int r=Discount(p);

int r1=Discount(r,8);

System.out.println("The discounted price is= "+r1);

}

}

The successive discounts are 10% and 8% which are hardcoded. The inputted price along with the discount is sent to the methods.
