What is a Rectangle?

To calculate volume and surface area of rectangle using member methods

Code For RectangleClassProg

public class Rectangle

{

double length; double breadth; double height;

Rectangle(double l,double b,double h)

{

length = l;

breadth = b;

height = h; } void surfacearea() {

double sa = 2 * (length * breadth + breadth * height + length * height);

System.out.println("Surface Area = " + sa); } void volume() {

double vol = length * breadth * height;

System.out.println("Volume = " + vol); } static void main() {

Rectangle sc=new Rectangle(20.0,17.0,16.0);

sc.surfacearea();

sc.volume();

}

}

using class surfacearea() we calculate surface area To find the surface area of a cuboid, add the areas of all 6 faces. We can also label the length (l), breadth (b), and height (h) and use the formula, SA=2lb+2lh+2hb, to find the surface area and through function volume we find the volume of cuboid that is l*b*h
