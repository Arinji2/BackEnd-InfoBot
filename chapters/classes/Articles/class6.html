Triplet

What is a Triplet?

When sum of square of two numbers forms a square of some other number, then these three numbers are called a Pythagorean triplet. Example:(3,4,5):

Code For Triplet

import java.util.*; class triplet { int a,b,c; void getdata() {

Scanner ob=new Scanner(System.in);

System.out.println("Enter three numbers");

a=ob.nextInt();

b=ob.nextInt();

c=ob.nextInt(); } void findprint() {

if(((a*a)+(b*b))==(c*c))

{

System.out.println("The numbers are a pythagorean triplet");

}

else

System.out.println("the numbers are not a pythagorean triplet"); } void main() {

triplet sc=new triplet();

sc.getdata();

sc.findprint(); } }

In this program we accept the input in void getdata().In void findprint() we check whether the sum of square of first two numbers is equal to the square of the third number. Accordingly we print appropriate statements. We use void main() to run the above functions.
