Factorial of a number

What is a Factorial of a number?

To find and print the factorial of the number as per the given specifications

Code For Factorial of a number

import java.util.*; public class FactorialData { private int n; void input() {

Scanner in = new Scanner(System.in);

System.out.println("Enter the number");

n = in.nextInt();

}

void fact() {

int i, factorial = 1;

for(i=1;i<n;i++)

{

factorial=factorial*i;

}

System.out.println("Factorial of "+n+" is: "+factorial);

} public static void main() { FactorialData ob=new FactorialData (); ob.input(); ob.fact();

} }

To find the factorial of the inputted number in the class FactorialData with the variable private int. Using the method void input() input the number. Then using the method void fact() calculate and print the factorial of the number. Finally in the main method create an object of the class and call all the methods.
