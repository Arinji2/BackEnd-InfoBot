Bill

How to Solve the Bill Program

It is a class as user defined type method for calculating the phone bill with a parameterized constructor

Code For Bill

import java.util.*;

public class Bill

{

int bno;

String name;

int call;

double amt;

Bill()

{

bno = 0;

name = "";

call = 0;

amt = 0.0;

}

Bill(int bno1, String name1, int call1)

bno = bno1;

name = name1;

call = call1;

}

public void calculate()

{

double charge;

if (call <= 100)

charge = call * 0.6;

else if (call <= 200)

charge = 60 + ((call - 100) * 0.8);

else if (call <= 300)

charge = 60 + 80 + ((call - 200) * 1.2);

else

charge = 60 + 80 + 120 + ((call - 300) * 1.5);

amt = charge + 125;

}

public void display()

{

System.out.println("Bill No: " + bno);

System.out.println("Name: " + name);

System.out.println("Calls: " + call);

System.out.println("Amount Payable: " + amt);

}

public static void main()

{

Scanner in = new Scanner(System.in);

System.out.print("Enter Name: ");

String custName = in.nextLine();

System.out.print("Enter Bill Number: ");

int billNum = in.nextInt();

System.out.print("Enter Calls: ");

int numCalls = in.nextInt();

Bill ob = new Bill(billNum, custName, numCalls);

ob.calculate();

ob.display();

}

}

In this program, the class name is named Bill as mentioned in the question. Data members like name, call, bno and amt are initialized in the parameterized constructor. public void calculate() method is used to find the charge and the total amount and is displayed in public void display(). In the main method, an object of the class is created and it calls the member methods.
