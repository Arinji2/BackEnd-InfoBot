Car

What is a Car Program?

A menu driven program to input showroom price and the number of years the car is used (‘1’ for one year old, ‘2’ for two years old and so on). Calculate the depreciated value. Display the original price of the car, depreciated value and the amount to be paid.

Code For Car

import java.util.*;

public class program

{

public static void main()

{

Scanner in = new Scanner(System.in);

int choice;

double price, dep = 0.0, amt;

System.out.println("1. One year old car");

System.out.println("2. Two year old car");

System.out.println("3. Three year old car");

System.out.println("4. Four year old car");

System.out.println("5. More than four year old car");

System.out.println("Enter your choice: ");

choice = in.nextInt();

System.out.println("Enter showroom price: ");

price = in.nextDouble();

switch(choice)

{

case 1:

dep = 0.1 * price;

break;

case 2:

dep = 0.2 * price;

break;

case 3:

dep = 0.3 * price;

break;

case 4:

dep = 0.5 * price;

break;

case 5:

dep = 0.6 * price;

break;

default:

System.out.println("Wrong choice!");

break;

}

amt = price - dep;

System.out.println("Original Price = " + price);

System.out.println("Depreciated Value = " + dep);

System.out.println("Amount to be paid = " + amt);

}

}

In the above program, the depreciated value is to be found. Being a menu driven program we use switch case with specifications mentioned in the question. the value is calculated with the total amount and displayed to the user
