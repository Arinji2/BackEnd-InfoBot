Next to Next Letter

What is a Next to Next Letter?

Replace each character with the next to next character. For Example A to C

Code For Next to Next Letter

import java.util.*; class Stringop { static String str; Stringop() {

str=""; } static void accept() {

Scanner ob=new Scanner(System.in);

System.out.println("Enter sentence");

str=ob.nextLine();

} static void encode() {

char b;

int i;

for(i=0;i<str.length();i++)

{

b=str.charAt(i);

if(b==' ')

System.out.print(b);

else if(b=='Y'||b=='y')

System.out.print("a");

else if(b=='z'||b=='Z')

System.out.print("b");

else

System.out.print(((char)(b+2)));

}

System.out.println(); } static void print() {

int i;

char b;

for(i=0;i<str.length();i++)

{

b=str.charAt(i);

if(b==' ')

System.out.println();

else

System.out.print(b);

} } static void main() {

Stringop sc=new Stringop();

sc.accept();

sc.encode();

sc.print(); } }

In this program we initialize the variable str to null in function Stringop().Then we take input in function void accept(). In void encode(),we use character extraction and print the next to next letter (i.e. for A we print C).In void print() we print each word of the string on a new line.Finally we use void main() to run the above functions.
