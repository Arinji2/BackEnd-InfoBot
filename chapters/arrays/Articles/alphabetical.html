Alphabetical Order

What is an Alphabetical Order?

To arrange 10 names input by the user in Alphabetical order.

Code For Alphabetical Order

import java.util.*;

class alphabetical order

{

static void main()

{

Scanner ob=new Scanner (System.in);

String m[]=new String[10];

int i,j;

String t;

System.out.println("Enter 10 names");

for(i=0;i<10;i++)

{

m[i]=ob.next();

}

for(i=0;i<9;i++)

{

for(j=0;j<9-i;j++)

{

if(m[j].compareToIgnoreCase(m[j+1])>0)

t=m[j];

m[j]=m[j+1];

m[j+1]=t;

}

}

}

System.out.println("Sorted Array:");

for(i=0;i<10;i++)

{

System.out.print(m[i]+",");

}

}

}

In this program, we arrange the names in alphabetical order by using bubble sort technique.Bubble sort technique is to swap the elements of the array depending on the condition given.
