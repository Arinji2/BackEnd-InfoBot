7 Wonders of the World(Linear Search)

What is a 7 Wonders of the World(Linear Search) ?

A program to assign 7 wonders of the world and input the name of the monument to be searched. If found display the name with its position(rank).

Code For 7 Wonders of the World(Linear Search)

import java.util.*;

public class program

{

public static void main()

{

int i;

String w[]= {"Taj mahal India" , "Machu Pichu Peru" , "Petra Jordan" , "Great Wall of China China" , "Chichen Itza Mexico" , "Colosseum Italy" , "Christ the Redeemer Brazil"};

Scanner in = new Scanner(System.in);

System.out.println("Enter the place to be found =");

String c = in.nextLine();

boolean x = false;

for( i = 0; i < w.length; i++)

{

if(c.equalsIgnoreCase(w[i]))

{

x = true;

System.out.println(" Rank = "+(i+1));

break;

}

}

if (x==false)

System.out.println("Invalid Input");

}

}

In this program, we hardcode the seven wonders in the order of their rank. When the user inputs the wonder whose rank they want to know, the program undergoes a linear search to find the wonder. In linear search loop, every element of the array is checked to match the condition given by the user until it find the correct element. if found, the loop exits by a break statement and the rank is displayed by adding +1 to the index position of the wonder and if the wonder is not there, the program states invalid input.
