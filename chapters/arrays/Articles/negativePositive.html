Negative Positive Sort

What is a negative positive sort

An array program to input 10 numbers in a Single Dimensional Array (SDA) and display all the negative numbers followed by the positive numbers without changing the order of the numbers.

Code For Odd and Even Sort

import java.util.*;

public class program

{

public static void main()

{

Scanner in = new Scanner(System.in);

int arr[] = new int[10];

System.out.println("Enter 10 numbers");

for (int i = 0; i < arr.length; i++)

{

arr[i] = in.nextInt();

}

for (int i = 0; i < arr.length; i++)

{

if (arr[i] < 0)

System.out.print(arr[i] + ", ");

}

for (int i = 0; i < arr.length; i++)

{

if (arr[i] >= 0)

System.out.print(arr[i] + ", ");

}

}

}

In this program, The number is sorted whether it is positive or negative. While printing the array, it undergoes a condition, if it is less than zero i.e. if it is negative, it will print else the loop will continue without changing its order. The second loop deals with positive numbers in the same way.
