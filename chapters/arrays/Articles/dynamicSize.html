What is a Search on number of elements input by user?

Write a program to input an array with the nth element taken by the user and a s number to be searched. If found display the message “search successful” with its position else “number not in the list”

Code For Search on number of elements input by user

import java.util.*;

class n_search

{

static void main()

{

Scanner ob=new Scanner (System.in);

System.out.println("Enter number of elements for the array");

int n=ob.nextInt();

int i;

int m[]=new int[n];

System.out.println("Enter numbers for the array");

for(i=0;i<n;i++)

{

m[i]=ob.nextInt();

}

int s;boolean x=false;

System.out.println("Enter number to be searched");

s=ob.nextInt();

for(i=0;i<n;i++)

{

if(s==m[i])

{

x=true;

break;

}

}

if(x)

System.out.println("Search successful at position ="+(i+1));

else

System.out.println("Number not in list");

}

}

For this search program,we take the input of the number of elements and use it to declare storage of the array and use that number for the limit of the for loop. Then we use linear search technique in which we element extraction and check whether is it equal to the search number
