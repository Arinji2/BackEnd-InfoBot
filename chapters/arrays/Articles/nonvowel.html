NonVowelNameArray

What is a NonVowelNameArray?

To input 10 names in a Single Dimensional Array and display the names that do not start with vowels

Code For NonVowelNameArray

import java.util.Scanner; public class array2 { public static void main() {

Scanner in = new Scanner(System.in);

String names[] = new String[10];

System.out.println("Enter 10 names");

for (int i = 0; i < names.length; i++)

{

names[i] = in.nextLine();

}

for (int i = 0; i < names.length; i++) {

if (Character.toUpperCase(names[i].charAt(0)) != 'A'&&Character.toUpperCase(names[i].charAt(0)) != 'E'&&Character.toUpperCase(names[i].charAt(0)) != 'I'&&Character.toUpperCase(names[i].charAt(0)) != 'O'&&Character.toUpperCase(names[i].charAt(0)) != 'U')

{

System.out.println(names[i]);

}

} }

}

Character.toUpperCase(names[i].charAt(0)) helps us extract the first character of the element of the array and check if its not equal to vowel if so then display it.
