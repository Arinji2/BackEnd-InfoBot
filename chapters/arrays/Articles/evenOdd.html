Even and Odd numbers

What is a Even and Odd numbers?

Write a program to accept 20 numbers in a Single Dimensional Array (SDA) arr[20]. Transfer and store all the even numbers in an array even[] and all the odd numbers in another array odd[]. Finally, print the elements of both the arrays.

Code For Even and Odd numbers

import java.util.*;

class even_odd_sort

{

static void main()

{

Scanner ob=new Scanner(System.in);

int m[]=new int [20];

int i,e=0,o=0;

System.out.println("Enter 20 number");

for(i=0;i<20;i++)

{

m[i]=ob.nextInt();

if(m[i]%2==0)

e++;

else

o++;

}

int even[]=new int[e];

int odd[]=new int[o];

int k1=0,k2=0;

for(i=0;i<20;i++)

{

if(m[i]%2==0)

{

even[k1]=m[i];

k1++;

}

else

{

odd[k2]=m[i];

k2++;

}

}

System.out.println("Even Array = ");

for(i=0;i<e;i++)

{

System.out.print(even[i]+" ");

}

System.out.println("Odd array = ");

for(i=0;i<o;i++)

{

System.out.print(odd[i]+" ");

}

}

}

In this program, while taking the input for 20 numbers we use counters and count the number of even and odd numbers to declare the storage of respective arrays. Then we do element extraction of an array and check whether they are an even number or an odd number and store it in the respective array.At the end we print both arrays.
