Grade List

What is a Grade List?

Write a program in Java using arrays: (a) To store the Roll No., Name and marks in six subjects for 100 students. (b) Calculate the percentage of marks obtained by each candidate. The maximum marks in each subject are 100. (c) Calculate the Grade as per given criteria

Code For Grade List

import java.util.*;

class grade_list

{

static void main()

{

Scanner ob=new Scanner(System.in);

String name[]=new String[100];

int rollno[]=new int[100];

int Bio[]=new int[100];

int Math[]=new int[100];

int Chem[]=new int[100];

int Comp[]=new int[100];

int Phy[]=new int[100];

int Eng[]=new int[100];

int i;

System.out.println("Enter student name,roll no. and marks for biology,physics,chemsitry,computers,maths and english");

for(i=0;i<100;i++)

{

name[i]=ob.nextLine();

rollno[i]=ob.nextInt();

Bio[i]=ob.nextInt();

Phy[i]=ob.nextInt();

Chem[i]=ob.nextInt();

Comp[i]=ob.nextInt();

Math[i]=ob.nextInt();

Eng[i]=ob.nextInt();

}

double percentage[]=new double[100];

char grade[]=new char[100];

for(i=0;i<100;i++)

{

percentage[i]=((double)(Bio[i]+Math[i]+Chem[i]+Comp[i]+Phy[i]+Eng[i])/6);

if(percentage[i]>=80.0 && percentage[i]<=100.0)

grade[i]='A';

else if(percentage[i]>=60.0 && percentage[i]<80.0)

grade[i]='B';

else if(percentage[i]>=40.0 && percentage[i]<60.0)

grade[i]='C';

else if(percentage[i]<40.0)

grade[i]='D';

}

System.out.println(" Student grade list:");

for(i=0;i<100;i++)

{

System.out.println("Student name: "+name[i]+" Roll number: "+rollno[i]+" Percentage= "+percentage[i]+" Grade: "+grade[i]);

}

}

}

In this program, we take the input for name,roll number and marks for respective subjects.Then we calculate percentage and use if else ladder to assign grades. Finally we print all the details.
