What is a Sum of Even and Odd?

Write a program to accept 10 numbers in a Single Dimensional Array(SDA) and find the sum of even and odd numbers.

Code For Sum of Even and Odd

import java.util.*;

public class array

{

static void main()

{

Scanner ob=new Scanner(System.in);

int m[]=new int[10];

int i,sum1=0,sum2=0;

System.out.println("Enter 10 numbers");

for(i=0;i<10;i++)

{

m[i]=ob.nextInt();

}

for(i=1;i<10;i++)

{

if(m[i]%2==0)

{

sum1=sum1+m[i];

}

else

{

sum2=sum2+m[i];

}

}

System.out.println("Sum of even numbers is "+sum1);

System.out.println("Sum of odd numbers is "+sum2);

}

}

After Inputting the Arrays, check through each element. If the element is divisble by 2, it is even else it is odd.
