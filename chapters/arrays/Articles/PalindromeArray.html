Palindrome Arrays

What is a Palindrome Arrays?

Palindrome arrays are a set of 2 arrays one of which has its first element same as the last element of the other array and so on.

Code For Palindrome Arrays

import java.util.*;

public class PalindromeArrays {

public static void main() {

Scanner in=new Scanner(System.in);

int a[]=new int[10];

int b[]=new int[10];

int i,k=0,t=9;

System.out.println("Enter the elements for Array1");

for(i=0;i<10;i++){

a[i]=in.nextInt();

}

System.out.println("Enter the elements for Array2");

for(i=0;i<10;i++){

b[i]=in.nextInt();

}

for(i=0;i<10;i++){

if(a[i]==b[t]){

k++;

t--;

}

}

if(k==10){

System.out.println("They are palindrome arrays");

}

else {

System.out.println("They are not palindrome arrays");

}

}

}

If the element at the 'i'th position of the first array is the same as that of the element at the 't'th position (of the other array), a counter 'k' is incremented and the value of t is decremented(this allows the program to check the second element from the end and so on). If the final value of k comes to 10, it signifies that all the elements in the two arrays are the same (but in opposite direction) and hence are palindrome arrays.
