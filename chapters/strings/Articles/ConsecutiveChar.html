ConsecutiveCharString

What is a ConsecutiveCharString?

Forming a new sentence by replacing each consonant with the previous letter. If the previous letter is a vowel then replace it with the next letter (i.e. if the letter is B then replace it with C as the previous letter of B is A). Sample Input – ICC WORLD CUP Sample Output – IBB VOQKC BUQ.
