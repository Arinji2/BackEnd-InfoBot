String replace

What is String replace?

To replace a given string to another string

Code For String Replace

import java.util.*;

public class prog12

{

static void main()

{

String s= " Keep the blue bottle in the blue bag kept on blue sofa.";

String s1= s.replace("blue","red");

}

}

.replace() replaces the string blue to red.
