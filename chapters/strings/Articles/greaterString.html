Greater string among the two strings

How to find the Greater string among the two strings?

Input two strings by using the string function .compareTo()

Code For finding Greater string among the two strings

import java.util.*;

public class compare

{

static void main()

{

Scanner in= new Scanner(System.in);

String str1 = in.nextLine();

String str2= in.nextLine();

System.out.println("String 1: " + str1);

System.out.println("String 2: " + str2);

int result = str1.compareTo(str2);

if (result < 0)

{

System.out.println( str1 + " is less than " + str2);

}

else if (result == 0)

{

System.out.println(str1 +" is equal to " + str2);

}

else

{

System.out.println( str1 +" is greater than " + str2 );

}

}

}

First we took the two strings and then used the function [.compareTo()] to check which string is the greater one. Then by converting it into explicit type (int) we showed the following results.
